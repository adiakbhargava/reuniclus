# Data Directory

Place the following files here before running Reuniclus with real connectome data:

## Connectome Adjacency Matrices (Phase 7)

- `connectome_adj.npy` — Channel-level adjacency matrix `[N × N]` float32.
  Constructed from the connectome synapse list via Phase 7.2.1 algorithm.
  Sources: FlyWire Consortium, Allen Mouse Brain, MICrONS, Eon Systems.

- `connectome_adj.csv` — Human-readable alternative for `ConnectomeGraph::from_csv()`.
  Format: `channel_i,channel_j,weight` (one synapse aggregate per line).

## Filter Reference Data (Phase 3 Validation)

Generated by `tools/compute_filter_coeffs.py`:

- `filter_input.npy` — Test signal (10 Hz + 60 Hz, 1 kHz, 100 samples).
- `filter_reference.npy` — scipy.signal.sosfilt output to compare against C++.

## Neural Session Data (Phase 4–7 Training)

- `neural_sessions.npy` — `[N, 256, window_size]` float32 neural recordings.
- `latent_states.npy`   — `[N, latent_dim]` float32 behavioural labels.

These are not included in the repository (patient data; research ethics).
Use `tools/train_tcn.py --synthetic` to generate synthetic replacements.
